Pop Quiz: Computer Science is a simple yet engaging Java-based quiz game designed to test your knowledge of fundamental Computer Science concepts.
The player progresses through five grade levels, each with increasingly challenging questions drawn from both theoretical and programming topics.

Features
Question Variety – A mix of theoretical and programming questions randomly selected from a 145-question pool.
Progressive Difficulty – Each grade level increases in question count and complexity.
Lives System – You start with three lives; each incorrect answer costs one.
Grade Progression – Successfully complete all five grades to win a Certificate of Achievement.
Simple UI – Custom-built Java Swing interface with a clean sky-blue gradient theme.
Audio Support – Optional background music and sound effects for correct and incorrect answers.
JSON Integration – Questions are loaded from an external questions.json file for easy editing.

How to Play
Launch the game.
Click Start to begin Grade 1.
Answer each question carefully — wrong answers reduce your lives.
Complete all questions in the current grade to advance to the next.
Lose all lives, and the game ends.
Complete all 5 grades to win!
